# Aula 04.B.ac - Expressões JOIN e Views

Este repositório contém a resolução da atividade **Aula 04.B.ac**, que aborda conceitos de **consultas SQL utilizando JOINs, subconsultas escalares e criação de Views** no banco de dados do esquema **University**.

A atividade foi desenvolvida utilizando **SQL Server** hospedado no **Azure SQL Database**.
